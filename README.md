# 🎵 Chinook Music Store Data Analysis  

## 📌 Project Overview  
This project analyzes the **Chinook Music Store Database** using SQL to uncover key business insights.  
The analysis covers **customer behavior, revenue trends, product performance, and regional sales patterns**.  
It demonstrates practical use of **SQL queries, joins, CTEs, subqueries, and window functions** for real-world data analysis.  

---

## ⚙️ Dataset Description  
The Chinook Database is a sample dataset commonly used for SQL practice. It contains:  
- **Customers** – customer details (location, contact info)  
- **Invoices & InvoiceLines** – sales transactions and track details  
- **Tracks / Albums / Artists / Genres** – music catalog data  
- **Employees** – support and sales representatives  

---

## 📊 Key Analysis Performed  
✔️ Customer acquisition vs repeat purchase rates  
✔️ Average order value (AOV) by region & customer  
✔️ Customer lifetime metrics (spend contribution)  
✔️ Sales analysis by track, artist, and genre  
✔️ Revenue distribution by billing country  
✔️ Cohort and retention analysis using **window functions**
